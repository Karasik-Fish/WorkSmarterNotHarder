# WorkSmarterNotHarder
На сайте pwn.college в блоке реверса есть задание Interoperability: Patching Data. По заданию требовалось пропатчить 4 байта а потом пройти игру. Мне показалось что тратить 10-20 минут на прохождение слишком долго. Писал бота несколько часов. Решение в лоб, громоздкое, работает с определенной вероятностью(тк генерация уровня не гарантирует кое-чего, но вероятность приятная). Зато в следующих 2 задачах тоже не пришлось играть
